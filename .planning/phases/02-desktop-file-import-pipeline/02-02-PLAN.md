  <read_first>
    - lib/features/import/presentation/import_screen.dart — 第58-62行 onDragEntered/onDragExited 为空操作回调
    - .planning/phases/02-desktop-file-import-pipeline/02-RESEARCH.md §5.5 — DragTarget + AnimatedContainer 虚线边框设计
    - .planning/phases/02-desktop-file-import-pipeline/02-CONTEXT.md D-03 — 桌面端拖放支持
  </read_first>

  <action>
    1. 将 `ImportScreen` 从 `StatelessWidget` 改为 `StatefulWidget`（需要跟踪拖放悬停状态）。

    2. 在 State 中添加 `bool _isDragOver = false` 状态变量。

    3. 修改 `_buildDesktopLayout` 方法中的 `DropTarget` 回调：
       ```dart
       onDragEntered: (_) {
         setState(() => _isDragOver = true);
       },
       onDragExited: (_) {
         setState(() => _isDragOver = false);
       },
       onDragDone: (details) {
         setState(() => _isDragOver = false);  // 拖放完成，隐藏覆盖层
         final filePath = details.files.firstOrNull?.path;
         if (filePath != null && _isSupportedFile(filePath)) {
           _navigateToProgress(context, filePath);
         } else if (filePath != null) {
           _showUnsupportedError(context);
         }
       },
       ```

    4. 在 DropTarget 的 child（Padding）外层包裹 Stack，添加拖放覆盖层：
       ```dart
       child: Stack(
         children: [
           // 主内容
           Padding(
             padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 24),
             child: Column(/* 现有内容 */),
           ),
           // 拖放覆盖层
           if (_isDragOver)
             Positioned.fill(
               child: AnimatedContainer(
                 duration: const Duration(milliseconds: 200),
                 decoration: BoxDecoration(
                   color: Theme.of(context).colorScheme.primaryContainer.withOpacity(0.15),
                   border: Border.all(
                     color: Theme.of(context).colorScheme.primary,
                     width: 2,
                     // Flutter 原生 Border 无虚线——用实线粗边框 + 圆角替代
                   ),
                   borderRadius: BorderRadius.circular(12),
                 ),
                 child: Center(
                   child: Column(
                     mainAxisSize: MainAxisSize.min,
                     children: [
                       Icon(
                         Icons.cloud_upload_outlined,
                         size: 48,
                         color: Theme.of(context).colorScheme.primary,
                       ),
                       const SizedBox(height: 12),
                       Text(
                         '释放以导入',
                         style: Theme.of(context).textTheme.titleMedium?.copyWith(
                           color: Theme.of(context).colorScheme.primary,
                           fontWeight: FontWeight.w600,
                         ),
                       ),
                       const SizedBox(height: 4),
                       Text(
                         '支持 .docx / .doc / .pdf / .json',
                         style: Theme.of(context).textTheme.bodySmall?.copyWith(
                           color: Theme.of(context).colorScheme.onSurface.withOpacity(0.7),
                         ),
                       ),
                     ],
                   ),
                 ),
               ),
             ),
         ],
       ),
       ```

    5. 确保覆盖层有 `IgnorePointer`（如果需要用户无法与覆盖层下方的元素交互）——但实际上拖放期间不需要交互，覆盖层作为视觉提示即可，不需要 IgnorePointer。

    6. 在 `_buildAndroidLayout` 中不需要拖放支持（Android 无桌面拖放），保持原样。
  </action>
