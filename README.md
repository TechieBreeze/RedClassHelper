# RedClass · 红课复习

本地化大学课程复习工具 — Flutter Windows 桌面 App。

> **平台支持**：目前**仅 Windows** 平台经过实际测试与可用验证。
> Android / iOS / macOS / Linux 仅完成基础编译通过，未做端到端测试。

## 已实现功能

- **题库导入**
  - PDF 文字题提取（pdfrx / PDFium）
  - DOCX 文字题提取（纯 Dart XML 解析，支持 Word 自动编号）
  - .doc 格式支持（pandoc 桥接转换）
  - 启发式解析器自动识别单选 / 多选 / 判断 / 简答
  - 解析器支持：章节标题自动断块、同行混合分隔符选项、独立行括号答案、题干内联答案
  - LLM 辅助解析（失败自动回退启发式兜底）
  - 解析结果可手动调整后入库
  - JSON 格式导入（跳过解析，直接入库）
- **刷题**
  - 题目导航、进度条、上一题 / 下一题
  - 多选题多选与取消选择
  - 键盘快捷键（A-H 选项、Space 确认、左右切题）
  - 答题会话持久化（退出后可恢复进度）
  - 错题标记
- **错题本**：自动收录错题，支持专项重做
- **统计**：正确率、按题型 / 模式分布
- **题库详情**：题库元信息、题目列表
- **导出**：题库导出为 JSON
- **主题**：Material 3 浅色 / 暗色完整支持，可切换预设主题色（青绿 / 蓝紫）
- **交互**：全局 hover 动效（缩放 + 阴影 + 手型光标）、选项卡点击反馈
- **设置**：所有页面可访问设置，主题模式 / 题库设置 / 模型管理
- **本地优先**：所有数据存 SQLite（drift），无后端依赖

## 技术栈

- Flutter 3.x · Dart
- Riverpod（状态管理）
- Drift / SQLite（本地存储）
- GoRouter（路由）
- pdfrx（PDF 提取）
- archive / xml（DOCX 提取）
