# RedClass (红课复习)

## What This Is

RedClass 是一个闭源的本地化大学课程复习工具。学生将老师下发的 `.docx` / `.pdf` 题库文件导入应用，由应用调用本地轻量 LLM 自动解析为结构化题目数据集；之后用户可在**乱序抽题**、**错题复习**、**错题抽查**三种模式间切换复习，三种模式间错题本互通联动（乱序产生的错题进入错题本，错题复习中答对的题目进入"已掌握"并从抽查范围中移除）。

**双层工作流**：
- **PC 端（Windows / macOS / Linux）**：完整能力，含 LLM 解析。可把解析后的题库**导出为标准 JSON 文件**。
- **移动端（Android / iOS）**：不含 LLM，只通过"导入 JSON 文件"路径获得题库（JSON 由 PC 端生成）。

同一套 Flutter 代码同时产出五个端。LLM 解析能力通过平台能力开关在 UI 层做差异（桌面端入口可见，移动端不渲染入口）。

## Core Value

把"老师发的题库文件"零摩擦地变成"可立刻投入复习的结构化题库"，让本地刷题体验比任何在线刷题网站都更顺手——**离线可用、零配置、解析即用、桌面解析、移动轻量**。

### Out of Scope

<!-- 显式排除：含排除理由，防止后续被反复提议 -->

- **云端同步 / 多设备同步** — 个人/小范围分享工具，无后端；JSON 文件就是用户可控的"传输协议"，足以覆盖跨设备需求
- **多用户登录 / 注册** — 无需账户，开箱即用，避免增加复杂度
- **题库中的图片/公式渲染** — 老师题库主要是文字，图片会引入 OCR + 资源管理复杂性，超出 v1
- **章节 / 知识点筛选** — 题库结构不一定有清晰章节标签，强加会引入错配
- **题目图片提取** — 与上一项配套排除
- **填空题 / 判断题 / 简答题** — 解析复杂度与判分逻辑不在 v1 范围
- **在线题库共享社区** — 闭源小工具，无社交/分享模块
- **暗色 / 浅色主题切换** — Material 3 默认主题即可，主题切换属 v2
- **字体大小调节** — 默认字号 + 平台原生缩放支持即可
- **移动端 LLM 解析** — 手机/iOS 内存、散热、电量约束下 1-3B 模型都不可靠；坚持 LLM 解析是 PC 端专属能力
- **跨设备错题本 / 收藏夹同步** — v1 只做"题库 JSON 单向传输"，错题本与收藏夹留在各端本地；如有需求 v2 再扩展

## Context

- **用户场景**：大学生期末复习，老师通常会以 `.docx` 或 `.pdf` 形式发放题库（含答案）；目前市面上多数刷题 App 都是为考研/公考设计的"题库已内置"模式，没有"导入自有题库"路径
- **痛点**：手动整理 Word/PDF 题库 → 复制粘贴到 Excel/Anki 耗时巨大；通用刷题 App 不能复用老师给的真实题库
- **目标受众**：本项目自用 + 同班同学分享（小范围），无需考虑规模化
- **运行环境**：开发者在 Windows / macOS / Linux 任一桌面端开发；用户分布在桌面端 + Android + iOS
- **设备差异化动机**：移动端内存 / 散热 / 电量约束使 1-3B 模型本地推理不可靠；用 JSON 文件作为跨设备的"传输协议"是更稳健的工程选择
- **UI 设计参考**：用户的 `ui-ux-pro-max` skill（`C:\Users\Lenovo\.claude\plugins\cache\ui-ux-pro-max-skill\ui-ux-pro-max\2.5.0`），在 UI 设计阶段作为视觉规范来源
- **已知技术决策**：Flutter 是核心，因为单代码库多端交付；PC 端本地 LLM 解析是关键，移动端通过 JSON 文件桥接

## Constraints

- **Tech stack**：Flutter（Dart）— 必须支持 Windows / macOS / Linux / Android / iOS 五端打包
- **本地化**：数据全部存在本地，无后端服务依赖
- **离线**：网络断开后所有功能仍可用（除首次下载 LLM 模型，PC 端专属）
- **闭源**：项目本身闭源，源码不公开发布；题库数据归用户所有
- **LLM 解析仅 PC**：仅在 Windows / macOS / Linux 上提供 LLM 解析；移动端只支持 JSON 导入
- **JSON 公开格式**：JSON 题库格式在 `doc/question-bank-json.md` 中明确定义，第三方可生成 / 校验
- **隐私**：用户的题库文件、错题、收藏不会离开本机（移动端从 PC 通过本地文件传输，不走云）
